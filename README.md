## Build
To build the place from scratch, use:

```bash
rojo build -o "Chaser.rbxlx"
```

Next, open `Chaser.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
